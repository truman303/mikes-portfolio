# Mike Fellowes Portfolio

I have an idea for my portfolio website. Since it's a personal website I thought I'd like to keep it simple, clean and fun by having a sketch theme, where the images paired with the text will be stick-figure drawings, doing typcial dev things like working on the whiteboard, working on a computer, looking at charts, fixing devices etc. In dark mode the stick figure should be white, in light mode it should be black.

## Layout

## Header

The top left corner should have a placeholder for a circular image of myself and subcopy with the following text: track record of delivery in utilities, system integrations, mobile apps, device integrations, point of sale, enterprise-ready software.

To the right corner should display menu items and a dark mode (sun/moon) button, icon-only.

The menu items are linked to the following subsections:

1. Mike
2. Work
3. Delivery
4. Brands
5. Contact

## Main Content

### Section 1: Mike Fellowes

Hero text is on the bottom left of the screen, reads "Hello, I'm Mike Fellowes".

A carousel of items with descriptions SVG stick figure icons should scroll across the screen from right to left just above the hero text in a loop. As the user mouses over the stick figure, the figure should animate to show the activity (in parenthesis).

- People Person (handshake)

### Section 2: Work

As the user scrolls down from section 1, section 2 should fade in and be fixed at the top.

The section should be titled "Work" and should be anchored at the tope the user scrolls through the section.

This section is made up of sub-sections per "domain" if you will.

Each sub-section should have a title on the left in a large font. On the right of each section are multiple stick figures illustrating the activity (as in section 1, except they are fixed in place). In each subsection, the title should fly in from the bottom of the screen and out the top of the screen as the users scrolls through the section. The stick figures should fade in and out on the right of the screen as the user scrolls past the subsection.

The sub sections titles and ideas for stick figures (in parenthesis) is listed below.

- Utilities (electricity, water, gas systems, system integrations)
- Payments (prepayment, point of sale, NFC, contact cards)
- IT management systems (multiple people working on computers)
- Defense & Radio Spectrum Management (radio transmitters, EW, radio waves, radio licence)
- Open source agent skills for the masses (AI agents, hand-crafted skills to help others)
